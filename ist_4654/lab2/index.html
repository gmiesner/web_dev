The Power of the Web

According to Tim Bernes-Lee , the inventor of the World Wide Web, at http://www.ww3.org/Press/IPO-announce:

The power of its web is in its universality. Access by everyone regardless of disability is an essential aspect.
TCP
Transmission Control Protocol is a method (protocol) used along with Internet Protocol (IP) to send data in form of message units, called packets, between computers over the Internet.
IP
Internet Protocol is the method or protocol by which data is sent from one computer to another on the Internt. Each computer on the Internet is uniquely identified by an IP address.
FTP
File Transfer Protocol is a protocol udes to exchange files between computers on the Internet.
HTTP
Hypertext Transfer Protocol is the protocol used for exchanging text, graphic images, sound, video, and other multimedia files on the Web.
